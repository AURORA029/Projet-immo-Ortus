1. OBJECTIFS STRATÉGIQUES

Créer une vitrine digitale d'exception pour l'agence ORTUS IMMOBILIER, capable de rivaliser avec les standards internationaux du luxe tout en s'adaptant aux contraintes locales (connexion, marché).

  • Positionnement : "Quiet Luxury" (Luxe discret et moderne).
  • Performance : Temps de chargement inférieur à 2 secondes.
  • Autonomie : Gestion 100% autonome sans développeur.

2. IDENTITÉ VISUELLE & UX

Une expérience utilisateur immersive basée sur les codes du luxe.

  • Charte Graphique : Noir Profond (#020610) & Or Métallique (#D4AF37).
  • Expérience Mobile : "Mobile First", intro animée (Splash Screen) et navigation fluide.
  • Typographie : Playfair Display (Titres) & Montserrat (Lecture).

3. FONCTIONNALITÉS CLÉS

  • Catalogue Dynamique : Synchronisation automatique avec Google Sheets.
  • Filtrage Intelligent : Catégorisation automatique "Prestige" vs "Classique".
  • Conversion : Formulaires contextuels avec redirection vers page de remerciement.
  • Acquisition Vendeurs : Page dédiée avec argumentaire commercial fort.

4. ARCHITECTURE TECHNIQUE (STACK MODERN)

Architecture "Serverless" garantissant sécurité et coût zéro.

  • Base de données : Google Sheets API (No-Code).
  • Frontend : HTML5 / CSS3 / Vanilla JS (Aucun framework lourd).
  • Hébergement : GitHub Pages (SSL Inclus).
  • Messagerie : Formspree (Gestion des leads par API).
