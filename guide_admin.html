ORTUS.

Manuel d'Administration du Site

Document technique réalisé par : RAFIDISON Ny Avo Andrianina Allan
Ce guide explique comment gérer les biens immobiliers du site via Google Sheets.
⚠️ RÈGLE D'OR - À LIRE AVANT TOUT
Ne jamais modifier, supprimer ou renommer la LIGNE 1 (L'en-tête bleu/gris) du fichier Google Sheets.
C'est elle qui fait le lien avec le site. Si vous changez un mot (ex: "prix" en "montant"), le site plantera.

1. ACCÉDER À L'INTERFACE DE GESTION

La gestion du site se fait entièrement via l'application Google Sheets (sur PC ou Mobile).

  • Ouvrez le fichier nommé : ORTUS_DB (Base de données).
  • Chaque ligne correspond à une maison affichée sur le site.

2. AJOUTER UNE NOUVELLE MAISON

Pour ajouter un bien, remplissez simplement une nouvelle ligne en bas du tableau.

Exemple de remplissage :
id
titre
prix
categorie
12
Villa Dorée Ivandry
400.000€
prestige

Détail des colonnes importantes :

  • id : Un chiffre unique (1, 2, 3...). Incrémentez à chaque fois.
  • titre : Le nom qui s'affiche en grand (ex: "Villa Panoramique").
  • image : Le lien URL de la photo (pas l'image elle-même).
  • prix : Le prix avec la devise (ex: "2.5 Mds Ar" ou "500.000 €").
  • ville : La localisation (ex: "Ivato", "Ambatobe").

3. GÉRER LE "PRESTIGE" (IMPORTANT)

Le site possède deux sections : Biens Classiques et Collection Prestige (Fond noir/or).

Comment classer un bien ?
Dans la colonne categorie :
  • Écrivez prestige (en minuscules) -> Le bien ira dans la page "Prestige".
  • Écrivez vente ou location -> Le bien ira dans la page "Biens" classique.

4. GÉRER LES IMAGES

Le fichier Excel ne peut pas stocker les images directement. Il faut utiliser un lien.

  1. Hébergez la photo (sur notre Drive public ou un site comme Imgur/Postimages).
  2. Copiez le Lien Direct de l'image (il doit finir par .jpg ou .png).
  3. Collez ce lien dans la colonne image du fichier Excel.

5. SUPPRIMER OU VENDRE UN BIEN

Option A : Marquer comme "Vendu"

Ne supprimez pas la ligne ! Cela montre notre succès.

  • Ajoutez simplement le mot (VENDU) à la fin du Titre.
  • Le site affichera toujours le bien, mais les clients sauront qu'il est parti.

Option B : Retirer du site

Effacez simplement toute la ligne correspondante dans le Google Sheet. Le bien disparaîtra du site instantanément (après rechargement).
